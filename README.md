# 기드온 협회 김포지회 홈페이지

본 프로젝트는 국제기드온협회 홈페이지(https://kgideons.org)의 공지사항 및 영상을 자동으로 크롤링하여 보여주는 프리미엄 홈페이지입니다.

## 🚀 시작하기

1. `index.html` 파일을 브라우저로 엽니다. (로컬 서버 권장)
2. 실시간 데이터 크롤링을 위해 아래 절차를 따르세요.

## 🔄 자동 크롤링 (데이터 업데이트)

본지(kgideons.org)의 최신 소식을 가져오려면 Node.js 환경에서 크롤러를 실행해야 합니다.

### 1. 필수 라이브러리 설치
터미널(또는 CMD)을 열고 프로젝트 폴더로 이동한 뒤 다음 명령어를 입력하세요:
```bash
npm install axios cheerio
```

### 2. 크롤러 실행
다음 명령어를 입력하면 `data/data.json` 파일이 최신 데이터로 업데이트됩니다:
```bash
node scripts/scrape.js
```

### 3. 자동화 팁
- **Windows 스케줄러**나 **Cron**을 사용하여 `node scripts/scrape.js`를 정기적으로(예: 매일 자정) 실행하도록 설정하면 완전 자동화가 가능합니다.

## ✨ 주요 기능
- **프리미엄 디자인**: 기드온 협회의 아이덴티티를 살린 세련된 UI.
- **실시간 소식 반영**: 본부의 간증 및 공지사항을 연동.
- **영상 선교 갤러리**: 미디어 게시판의 최신 영상을 홈에서 바로 확인.
- **모바일 최적화**: 반응형 웹 디자인으로 스마트폰에서도 완벽하게 작동.
